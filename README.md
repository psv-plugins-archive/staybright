# Staybright

Disable screen auto-dimming but retain auto-suspend on the Playstation Vita.

## Installation

Supported firmware versions:

- Retail 3.60-3.73

Install under `*KERNEL` of your taiHEN config.

```
*KERNEL
ur0:tai/staybright.skprx
```
